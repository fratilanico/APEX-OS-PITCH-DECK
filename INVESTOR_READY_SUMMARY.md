## HOW TO USE

### View in Browser
```bash
cd /home/ubuntu/APEX-OS-PITCH-DECK/pitch-deck
python3 -m http.server 8765
# Open: http://localhost:8765/index_v4_investor_ready.html
```

### Navigate
- **Arrow keys** or **click** to navigate
- **Space** to advance
- **Home/End** to jump to first/last slide

### Export to PDF
1. Open in Chrome/Chromium
2. Press Ctrl+P (or Cmd+P on Mac)
3. Select "Save as PDF"
4. Set margins to "None"
5. Enable "Background graphics"
6. Save as `APEX_OS_Pitch_Deck.pdf`

# AIAppBuddy Features (from LinkedIn post by Daniel Berk)

## Description
"The Duolingo of AI, but for working professionals"

## Target Audience
Working professionals who know they should use AI more but don't know where to start.

## Learning Tracks (Roles/Personas)
1. Sales
2. Engineering
3. Marketing
4. Founders & Execs
5. Productivity

## Key Features
- Seven learning modules per track
- Role-specific prompts and templates (30 per track)
- Hands-on exercises with real AI tools
- Prompt template library
- Practice playground to build skills
- Progress tracking
- Simple onboarding flow
- Curated learning path based on profile

## Core Concept
Training platform that teaches how to use AI for YOUR specific job role.
"Think Duolingo, but for AI skills — and built for people with real jobs."
